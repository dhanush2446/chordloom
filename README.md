# Invisible Theremin Pro

A hand-tracking theremin instrument using MediaPipe for gesture recognition. Play music by moving your hands in front of the camera.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
